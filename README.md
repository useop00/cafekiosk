# Cafe Kiosk Project

이 프로젝트는 카페 키오스크 API를 제공하기 위한 Spring Boot + Kotlin 애플리케이션입니다.

- [API 문서 바로가기](https://useop00.github.io/cafekiosk/)

---

## 프로젝트 소개
카페 키오스크에서 판매되는 **COFFEE, BOTTLE, DESSERT** 등 상품 정보를 관리하고, 주문을 처리하는 백엔드 API입니다.

## 개발 환경
- **Language**: Kotlin
- **Framework**: Spring Boot 3.4.1
- **DB**: H2
- **Build**: Gradle
